# 🚀 Manoj Dahal | Portfolio

A modern, responsive, and beautifully designed portfolio website for **Manoj Dahal**, Full Stack Developer & UI/UX Designer.

Built with clean HTML5, CSS3, and Vanilla JavaScript. Features stunning visuals, smooth animations, dark mode, interactive particles, and a professional layout.

![Portfolio Preview](https://raw.githubusercontent.com/manoj-dahal/portfolio/refs/heads/main/images/profile.jpg)

## ✨ Design Highlights (2025 Update)

- **Stunning New Visuals**: Fresh AI-generated high-quality project mockups, professional profile photo, and beautiful hero background
- **Modern Hero Section**: Elegant full-bleed background with subtle overlay + interactive particle system
- **New Sections**: Featured **Case Studies** (with metrics) + enhanced **Testimonials Carousel**
- **Premium UI/UX**: Refined typography, better spacing, improved card designs
- **Color Refresh**: Teal primary palette (`#14b8a6`) + increased card gaps
- **Responsive Design** — Works seamlessly on all devices
- **Dark/Light Mode** — Toggle with persistent storage (dark default)
- **Interactive Particle Background** — Beautiful animated particles on hero
- **Advanced Animations**: Magnetic buttons, parallax hero, scroll-triggered effects, carousel, confetti on form success
- **Project Filtering** — Dynamic category filtering
- **Improved Contact Form** — Real-time validation, success state, confetti celebration
- **SEO Optimized** — Proper meta tags, Open Graph, and semantic HTML
- **Accessibility** — WCAG compliant with ARIA labels, skip link, reduced-motion support, keyboard/touch carousel

## 🛠️ Technologies Used

- HTML5 + CSS3 (CSS Variables, Flexbox, Grid, Modern Layout)
- Vanilla JavaScript (ES6+)
- AOS (Animate On Scroll)
- Font Awesome Icons
- Google Fonts (Poppins + Fira Code)
- Canvas Particle System

## 🚀 Quick Start

1. Clone the repository
2. Open `index.html` directly in a browser (no build step needed)
3. Or deploy to GitHub Pages / Vercel / Netlify
